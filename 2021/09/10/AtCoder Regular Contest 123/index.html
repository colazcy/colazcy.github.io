A.Arithmetic Sequence

题意：给定 \(a,b,c\)，每次操作为 \(a \leftarrow a + 1\)，或 \(b \leftarrow b + 1\)，或 \(c \leftarrow c + 1\)。求最小的操作次数使得 \(b - a = c - b\)。

\(a,b,c \leq 10^{15}\)。

2s，1024M。

移项可得 \(2b = a+c\)。

先操作 \(b\) 使得 \(2b \geq a+c\)，再操作 \(a,c\) 即可。

时间复杂度 \(\Theta(1)\)。

B.Increasing Triples

题意：给定三个序列 \(a,b,c\)，你可以将它们任意重排，使得 \(a_i < b_i < c_i\) 的位置 \(i\) 尽量多。求最多的位置数。

\(1 \leq n \leq 10^5\)，

\(1 \leq a_i,b_i,c_i \leq 10^9\)。

2s，1024M。

因为可以任意重排，所以转化为选取三元组。根据决策包容性，我们肯定贪心选最小的 \(a_i\)，\(a_i\) 确定之后选择可能的最小 \(b_i\)，然后选择可能的最小 \(c_i\)。

排好序之后直接二分查找。

C.1, 2, 3 - Decomposition

题意：给定 \(n\)，求最小的 \(k\)，使得存在一个正整数序列 \(a_1,a_2,\cdots,a_k\)，使得 \(\sum_{i=1}^k a_i=n\)，且 \(a_i\) 十进制表示下的每一位属于 \(\{1,2,3\}\)。

\(t\) 组询问。

\(1\leq t \leq 1000\)。

\(1\leq n \leq 10^{18}\)。

2s，1024M。

答案不会太大，我们可以枚举 \(k\) 并检验。归纳法可以证明答案不超过 \(5\)，题解给出了证明。

从低位往高位考虑。发现只要该位不为 \(0\) 的 \(a_i\) 个数确定了，更低位对它的进位确定了，那么可以确定该位的选数方案。

因此我们可以记 \(f[step][now][cnt]\) 表示当前是第 \(step\) 位，有 \(now\) 位不为 \(0\)，低位的进位为 \(cnt\)。

枚举 \(i \in [now,3now]\)，若 \(i + cnt \equiv n_{step} \pmod {10}\)，则转移到 \(f[step + 1][nxt][\lfloor \frac{i + cnt}{10}\rfloor] \quad nxt \leq now\)，直接记忆化搜索。

D.Inc, Dec - Decomposition

题意：给定长为 \(n\) 的序列 \(a\)，构造长为 \(n\) 的序列 \(b,c\)。满足：

  • \(a = b_i + c_i\)。

  • 序列 \(b\) 单调不降。

  • 序列 \(c\) 单调不升。

\(1 \leq n \leq 2\times 10^5\)。

\(-10^8 \leq a_i \leq 10^8\)。

2s，1024M。

似乎 Slope Trick 相较于线性做法更为通用？

可以在 USACO，zscoder's blog，Kuroni's blog 找到相关资料。

简言之，Slope Trick 可以用来表示满足以下特征的函数：

  • 连续
  • 是凸函数（即斜率单调）
  • 可以分为若干段，每段均为一次函数

我们可以用一个可重集 \(S\) 以及一个一次函数 \(y = kx + b\) 来表示一个满足以上特征的函数，可重集 \(S\) 内的每个位置 \(p\) 表示在 \(p\) 点斜率改变 \(1\)（斜率可能改变不止 \(1\)，所以是个可重集） 。\(y = kx + b\) 表示该函数最靠右的一段是一次函数 \(y = kx + b\)。

有了 \(y = kx + b\)，我们可以从大到小遍历 \(S\)，从而确定函数每一段。

利用 Slope Trick，我们可以方便的表示出 \(F(x) + G(x)\)，其中 \(F(x),G(x)\) 都满足以上特征并且它们都是上（下）凸的。二者最右段一次函数直接相加，两个可重集直接合并。（类似于扫描线，每个元素都代表斜率改变 \(1\) 这个事件，所以可以直接合并）。

对于这道题，题意可以转化为，给定序列 \(a,b\)。求序列 \(x\)，在 \(x_i + a_i \leq x_{i+1}\) 的前提下，最小化 \(|x_i|+|x_i-b_i|\)。

有平方级别的 dp：记 \(f_i[x]\) 表示考虑序列 \(x\) 的前 \(i\) 项，并且 \(x_i = x\)，答案的最小值。\(g_i\) 为 \(f_i\) 的前缀 \(\min\)。

那么有 \(f_i[x] = g_{i-1}[x-a_i]+|x|+|x-b_i|\)。

显然 \(|x| + |x-b_i|\) 符合上述特征，对于函数平移直接整体打标记即可。

E.Training

题意：给定 \(A,B,C,D,n\)。求 \(\sum_{x=1}^{n}[(A+\lfloor\frac{B}{x}\rfloor) = (C+\lfloor\frac{D}{x}\rfloor)]\)。

\(t\) 组询问。

\(1 \leq t \leq 2\times10^5\)。

\(1 \leq n \leq 10^9\)。

\(1 \leq A,B,C,D \leq 10^6\)。

3s，1024M。

记 \(f(x) = A + \frac{B}{x},g(x)=C+\frac{D}{x},F(x)=\lfloor f(x)\rfloor,G(x) = \lfloor g(x)\rfloor\)。

若 \(F(x) = G(x)\)，一定有 \(|f(x)-g(x)| \leq 1\)。

一次函数交点容易计算。

假设 \(0 \leq f(x)-g(x) \leq 1\)，另一种情况同理。那么 \(F(x)=G(x)\) 或 \(F(x)=G(x)+1\)。我们现在要求 \(\sum_{x=l}^r[F(x)=G(x)]\)。有：

\[ \begin{aligned} \sum_{x=l}^{r}[F(x)=G(x)]&=(r-l+1)-\sum_{x=l}^r[F(x)=G(x)+1]\\ &=(r-l+1)-\sum_{x=l}^r(F(x)-G(x)) \end{aligned} \] 求 \(\sum_{x=l}^rF(x)\)，等价于求 \(\sum_{x=0}^nF(x)\)，可以 \(\Theta(1)\) 求出。

F.Insert Addition

题意：对于序列 \(p = (p_1,p_2,\cdots,p_m)\)，定义函数 \(f(p) = (p_1,q_1,p_2,q_2,\cdots,p_{m-1},q_{m-1},p_m)\)，其中 \(q_i=p_{i}+p_{i+1}\)。给定的序列 \(A = (p,q)\)，每次令 \(A \leftarrow f(A)\)，重复 \(n\) 次。之后删去所有大于 \(n\) 的元素得到序列 \(B\)。给定 \(l,r\)，求 \(B_l,B_{l+1},\cdots B_r\)。

\(1 \leq n \leq 3\times 10^5\)。

\(1 \leq a,b \leq n\)。

\(1 \leq l \leq r \leq 10^{18}\)。

\(r - l < 3 \times 10^5\)。

4s，1024M。

前置知识：Stern-Brocot 树与 Farey 序列。

初始序列为 \(\frac{1}{0},\frac{0}{1}\)。

每次在相邻两个分数 \(\frac{a}{b},\frac{c}{d}\) 中间插入分数 \(\frac{a + c}{b + d}\)。

一些性质：

  • 单调性

容易证明，类似于溶液混合，有 \(\frac{a}{b} < \frac{a + c}{b + d} < \frac{c}{d}\)。

  • 最简性

序列中的分数均为最简分数。

归纳法可以证明对于任意连续分数 \(\frac{a}{b},\frac{c}{d}\)，有 \(bc - ad = 1\)。

因此根据裴蜀定理，上述方程有解，因此 \(\gcd(a,b) = \gcd(c,d) = 1\)。

原问题与之十分类似。由于 \(n\) 次操作后新得到的数一定大于 \(n\)，所以我们可以忽略只操作 \(n\) 次这个限制。对于相邻两个数 \(p,q\)，它们之间的数一定可以表示为 \(px + qy \quad \gcd(x,y) = 1\) 的形式。

现在我们可以把这棵树当做线段树，问题变成，对给定的 \(p,q,n\)，求 \(\gcd(x,y) = 1,px + qy \leq n\) 的 \((x,y)\) 的对数。

使用莫比乌斯反演，不考虑 \(\gcd(x,y)=1\) 的限制容易处理，记其为 \(g(n)\)。记考虑 \(\gcd(x,y) = 1\)的限制时的答案为 \(f(n)\)。

有 \(g(n) = \sum_d f(\lfloor \frac{n}{d}\rfloor)\)。

所以有 \(f(n) = \sum_d \mu(d) g(\lfloor \frac{n}{d}\rfloor)\)。
